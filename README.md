## Simple-Bluetooth-Positioning-System

利用樹梅派與藍牙模組實現三角定位

## <概述>

使用一個藍牙模組持續發送訊號，以樹梅派的作基地台偵測藍牙模組的訊號並回傳給主控台，透過訊號的強弱計算出與基地台間的距離。
透過測量與三個基地台間的距離，定位出藍牙模組的所在地。

## <分工>

文件製作、上傳：郭晉廷、魏志穎

主程式：趙守恩

## <素材>

1 . Raspberry pi 3 * 3

2 . 藍牙模組 * 1

3 . 各式線材
